# Face Recognition System

A real-time face recognition system using OpenCV and a pre-trained deep learning model. This project demonstrates how to build a system that can identify individuals from a live video stream.

## Features

*   **Real-Time Performance**: Processes video streams from a webcam in real-time.
*   **Deep Learning Model**: Uses a pre-trained deep learning model for accurate face detection and recognition.
*   **OpenCV Integration**: Built with OpenCV for efficient video capture and image processing.

## Getting Started

### Prerequisites

*   Python 3.8+
*   OpenCV
*   A webcam

### Installation

```bash
git clone https://github.com/Thatimmorsit/face-recognition-system.git
cd face-recognition-system
pip install -r requirements.txt
```

### Usage

```bash
python recognize.py
```
